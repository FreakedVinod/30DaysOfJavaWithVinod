## 📌 Topics Covered

- String Traversal

## 💻 Programs Implemented

### 1. Count Vowels
- Counted number of vowels in a string
- Used character array and loop traversal
- Handled both uppercase and lowercase input

### 2. Password Strength Checker
- Validated password based on rules:
  - Minimum length (8 characters)
  - At least one uppercase letter
  - At least one lowercase letter
  - At least one digit
- Used Character class methods for validation

## 💡 Key Learnings
- Understanding character classification in Java
- Writing validation logic for real-world scenarios
- Using built-in methods like Character.isUpperCase()
