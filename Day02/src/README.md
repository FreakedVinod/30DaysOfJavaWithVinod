# Day 2 – Java + DSA Challenge 🚀

## 📌 Topics Covered
- Linear Search Algorithm
- User Input Handling using Scanner
- Loop and Conditional Logic
- Basic Game Development in Java

---

## 💻 Programs Implemented

### 1. Linear Search
- Takes dynamic input from user
- Searches for a target element in an array
- Displays the index of the element if found

### 2. Number Guessing Game
- Generates a random number
- User guesses the number
- Provides hints ("Too High" / "Too Low")
- Continues until correct guess
